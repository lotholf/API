API PMSIpilot

Identités/Mouvements

Informations en temps-réel sur les identités, séjours et mouvements de l'établissement
Resource Description
GET api/identites Retourne les identités.

Chaque identité contient ses propres informations ainsi que l'ensemble des séjours associés.
Chaque séjour associé contient ses propres informations ainsi que l'ensemble des mouvements associés.
D'autres API sont disponibles pour accéder direct
GET api/sejours Retourne les séjours.

Chaque séjour contient ses propres informations, celles de l'ensemble des mouvements associés et celles de son patient.
D'autres API sont disponibles pour accéder directement aux patient ou aux mouveme
GET api/mouvements Retourne les mouvements.

Chaque mouvement contient ses propres informations, celles de son séjour et celles de son patient.
D'autres API sont disponibles pour accéder directement aux patient ou aux sejours.
