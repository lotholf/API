API PMSIpilot

La puissance et l'expertise de PMSIpilot à la disposition de tous.

Les API PMSIpilot permettent d'accéder en temps-réel aux informations du SIH.
Les identités, séjours, actes et diagnostics deviennent facilement accessibles.

Les API PMSIpilot proposent l'utilisation du groupeur pour le MCO et le SSR.
Les RSS et RHS peuvent être générés à partir des informations du SIH.

Les API PMSIpilot offrent la possibilité de vérifier la qualité du codage PMSI.
L'ensemble des tests Qualifact sont ouverts à une utilisation externe.

Les API PMSIpilot valorisent les RSS et RHS.
Il est ainsi possible d'obtenir l'activité valorisée de l'établissement.

Les API PMSIpilot sont disponibles pour tous les clients PMSIpilot.
Elles peuvent être utilisées par n'importe quel logiciel tiers. Quel qu'en soit l'éditeur.
Le département informatique de l'établissement peut aussi les utiliser pour des applications "maison".
