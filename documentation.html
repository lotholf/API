API PMSIpilot

Les API PMSIpilot utilisent le protocole HTTP basé sur des requêtes et des réponses.

Liste des API disponibles

Identités

Informations en temps-réel sur les identités.
Resource Description
GET api/identites Retourne la liste des identités.
GET api/identites/:ipp Retourne l'identité identifiée par :ipp.
POST api/identites Ajoute une identité.
PUT api/identites/:ipp Modifie l'identité identifiée par :ipp.
PUT api/identites/:ipp-destination/merge/:ipp-source Fusionne l'identité identitfiée par :ipp-source dans l'identité identifiée par :ipp-destination.
DELETE api/identites/:ipp Supprime l'identité identifiée par :ipp.
Séjours

Informations en temps-réel sur les séjours.
Resource Description
GET api/sejours Retourne la liste des séjours.
GET api/identites/:ipp/sejours Retourne la liste des séjours de l'identité identifiée par :ipp.
GET api/sejours/:iep Retourne le séjour identifié par :iep.
POST api/identites/:ipp/sejours Ajoute un séjour à l'identité identifiée par :ipp.
PUT api/sejours/:iep Modifie le séjour identifié par :iep.
DELETE api/sejours/:iep Supprime le séjour identifié par :iep.
Mouvements

Informations en temps-réel sur les mouvements.
Resource Description
GET api/mouvements Retourne la liste des mouvements.
GET api/sejours/:iep/mouvements Retourne la liste des mouvements d'un séjour identifié par :iep.
GET api/mouvements/:id Retourne le mouvement identifié par :id.
POST api/sejours/:iep/mouvements Ajoute un mouvement au séjour identifié par :iep.
PUT api/mouvements/:id Modifie le mouvement identifié par :id.
DELETE api/mouvements/:id Supprime le mouvement identifié par :id.
